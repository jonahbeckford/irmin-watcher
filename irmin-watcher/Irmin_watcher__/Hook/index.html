val v : wait_for_changes:(unit -> event Lwt.t) -> dir:string -> Irmin_watcher__.Core.Watchdog.hook

v ~wait_for_changes ~dir is the watchdog hook using wait_for_changes to detect filesystem updates in the directory dir. The polling implemention just calls Lwt_unix.sleep.
